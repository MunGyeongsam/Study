#### 📚 Logger 모듈 API 사용법
```lua
-- 모듈 로드 (현재 lib 폴더에 위치)
local Logger = require("lib.logger")

-- 초기화
Logger.init()                    -- 로깅 시스템 초기화 (로그 파일 생성)

-- 로깅 레벨별 사용법
Logger.debug("Player pos: " .. x .. "," .. y)  -- 회색 디버그 정보
Logger.info("Game started")                   -- 녹색 일반 정보
Logger.warn("Performance issue detected")     -- 노란색 경고
Logger.error("Critical error occurred")       -- 빨간색 오류

-- 인게임 콘솔 관리
Logger.addToConsole("Custom debug info")      -- 콘솔에 메시지 추가
local messages = Logger.getConsoleMessages()  -- 콘솔 메시지 조회

-- 종료 시 정리
Logger.close()                   -- 로그 파일 닫기
```

### ⚡ 성능 & 최적화 정보 (Performance & Optimization)

#### 현재 성능 상태
- **해상도**: 432x960 (9:20 portrait) → 비교적 경량
- **목표 FPS**: 60 FPS (love.timer.getFPS()로 실시간 모니터링)
- **메모리 사용**: 기본 상태 → 대량 엔티티 대비 필수
- **객체 수**: 현재 1개 (player) → 1000+ 목표

#### LOVE2D 최적화 전략 (Planning)
```lua
-- 객체 풀링 패턴 (계획)
local bulletPool = {}           -- 재사용 가능한 탄환 풀
local activeCount = 0           -- 활성 객체 수 추적

-- 배치 렌더링 (계획)
love.graphics.push()            -- 그래픽 상태 저장
-- 동일 색상/형태 객체 일괄 렌더링
love.graphics.pop()             -- 상태 복원

-- 메모리 최적화
local reusableVec2 = {x=0, y=0} -- 단일 벡터 재사용
collectgarbage("collect")       -- 수동 GC (디버깅용)
```

### 🔍 디버깅 & 컨트롤 (Debug & Controls)

#### 키 바인딩 (사용법)
- **F1**: 디버그 콘솔 토글 (`debugConsoleVisible = !debugConsoleVisible`)
- **F2**: 로깅 테스트 (DEBUG/INFO/WARN/ERROR 모두 출력)
- **F3**: 디버그 모드 토글 (player 위치 시각화, gameSpeed 2.0)
- **ESC**: 게임 종료
- **WASD/Arrow Keys**: 플레이어 이동 (계획됨)

#### 디버그 정보 확인법
```lua
-- 게임 내 콘솔로 디버그 메시지 출력
addDebugMessage("Custom info: " .. someValue)

-- 실시간 게임 상태 확인
print("gameTime: " .. gameTime)           -- 에러 품질 사용 X
Logger.debug("gameTime: " .. gameTime)    -- 권장 사용

-- FPS 모니터링
local fps = love.timer.getFPS()
if fps < 50 then
    Logger.warn("Low FPS detected: " .. fps)
end
```

#### 디버그 콘솔 UI
- **위치**: 화면 상단 50% 차지 반투명 배경
- **색상**: 녹색 텍스트, 노란색 안내 메시지
- **메시지 제한**: 최대 15개 라인 (자동 스크롤)

### 🚫 자주 발생하는 이슈 (Troubleshooting)

#### 성능 이슈
- **느진 FPS**: 너무 많은 엔티티 생성 → 객체 풀링 필수
- **메모리 누수**: 임시 객체 근본적 제거 → 재사용 패턴
- **GC 지연**: table.insert/remove 남용 → 고정 배열 사용

#### LOVE2D 주의사항
- **로컬 변수**: love.update() 내에서 local 선언 지양
- **계산 비용**: math.sin/cos 반복 계산 → 룩업 테이블 사용
- **문자열 연결**: .. 연산자 남용 → string.format 사용
