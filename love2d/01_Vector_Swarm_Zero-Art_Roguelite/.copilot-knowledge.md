# Vector Swarm — Knowledge Base

> Copilot 자동 참조용 프로젝트 요약. 상세 API/구조는 `src/SOURCE_MAP.md` 참조.

## 프로젝트 정체성

- **장르**: 세로 슈팅 + 로그라이트 (뱀서라이크 + 탄막)
- **철학**: Zero-Art — 외부 에셋 없이 코드만으로 모든 그래픽/사운드 생성
- **기술**: LÖVE2D 11.5 / Lua 5.1, ECS 아키텍처, Scene Stack
- **해상도**: 432×960 (9:20 portrait)

## 현재 상태

- **Phase 0~5 완료** (기반→MVP→확장→콘텐츠→폴리싱→엔드게임)
- **Phase 6 진행 중** — 절차적 적 DNA 시스템 (6B 완료, 6A/6C/6D 진행 중)
- **90개 Lua 파일**, ~12,000 lines
- **ECS**: 17 컴포넌트, 18 시스템, bulletPool(ECS 외부)

## 아키텍처 요약

```
레이어: 00_common → 01_core → 02_renderer → 03_game → 04_ui / 05_sound
        (높은 번호가 낮은 번호를 require; 역방향 금지)
렌더:   bloom.begin → camera.draw(ECS) → bloom.end → UI
입력:   touch/mouse → sceneStack → top scene → uiManager or game
```
