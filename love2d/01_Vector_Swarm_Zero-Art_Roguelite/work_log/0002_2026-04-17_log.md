- OOP player → ECS 파사드 패턴으로 전환

- `InputSystem` — 터치/마우스 입력 → velocity 컴포넌트 반영
- `BoundarySystem` — 월드 경계 제한
- `PlayerRenderSystem` — 삼각형 + 방향 표시 렌더링 (lineWidth 설정)
- `player.update()` — 존 감지/파워업 등 월드 인터랙션만 담당
- `player.draw()` — 비어있음 (PlayerRenderSystem이 처리)

#### 6. Logger 정리 ✅

- `logger.error()` 중복 코드 제거 → `writeLog()` 내부에서 ERROR 시 stderr 추가
- `local err` 전역 누출 수정
- 이모지 로그 → ASCII 태그로 변환 (`[CAM]`, `[ECS]`, `[ENTITY]`, `[PLAYER]`, `[ZONE]`, `[WORLD]`)
- `init()` 초기 메시지 5개 → `logger.info()` 1줄로 간결화
- `drawConsole()` 키 안내 → 현재 바인딩 반영
- 콘솔 기본값 hidden (main.lua에서 toggleConsole 호출 제거)
- 260줄 → ~130줄 (50% 감소)
