## 3B.3 고급 탄막 — orbit_shot + return_shot ✅

**가치 기반 설계 결정:**
- 5개 후보 (homing, accel, orbit_shot, sine_wave, return_shot) 중 3원칙(재미>쾌감>제약의미학) 기준 평가
- orbit_shot (10/10): 공전→사출 = 예측 불가 × 아름다움, 수학적 궤적
- return_shot (10/10): 감속→반전 = 이중 위험, 공간 재활용의 재미
- homing: 7/10 — 재미는 있지만 탄막 게임 정체성과 거리감 → 3B.4로 보류

**bulletPool behavior 시스템:**
| 필드 | 용도 |
|------|------|
| `behavior` | nil(등속직선) / "orbit" / "return" |
| `originX/Y` | orbit: 공전 중심 |
| `orbitRadius/Speed/Angle/Time` | orbit: 반경, 각속도, 초기각, 공전시간 |
| `returnTime` | return: 감속→반전 타이밍 |
| `returned` | return: 반전 여부 플래그 |

**orbit_shot 동작:**
1. 발사 시 emitter 위치 주위 원 위에 배치
2. `lifetime < orbitTime` → cos/sin으로 위치 직접 계산 + 접선 속도 갱신
3. `orbitTime` 경과 → 접선 방향 등속 사출 (기본 이동)

**return_shot 동작:**
1. 발사 시 damping=0.15 (강한 감속)
2. `lifetime >= returnTime` → 속도 반전 × 1.3배 + damping=1 해제
3. 부메랑처럼 되돌아옴

**보스 적용:**
| 보스 | 페이즈 | 패턴 | 설정 |
|------|--------|------|------|
| HEAP | P3 | orbit_shot | 6발, 반경1.2, 각속도3.5, 공전1.0초 |
| OVERFLOW | P4 | return_shot | 8발, 반전0.7초, damping 0.15 |

**homing 보류:** 로드맵 3B.4로 이동, bulletEmitter 패턴 목록에 "(planned: homing)" 기록
