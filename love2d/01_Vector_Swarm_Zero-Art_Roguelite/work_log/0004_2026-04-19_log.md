#### 1. Bloom 포스트프로세싱 셰이더 (3C.1) ✅

**설계 결정:**
| 결정 | 선택 | 이유 |
|------|------|------|
| Bloom 대상 | 월드 전체 (카메라 영역) | 어두운 배경 + 밝은 오브젝트 구조라 threshold만으로 충분 |
| Blur 해상도 | 절반 (216×480) | 부드러운 glow + GPU 부하 절감 |
| 파일 구조 | 독립 모듈 `bloom.lua` | 카메라와 책임 분리, on/off 토글 용이 |

**파이프라인:**
```
beginCapture() → sceneCanvas에 월드 렌더링
endCapture()   → 캡처 종료
draw():
  1. sceneCanvas → threshold 셰이더 → brightCanvas (밝은 픽셀 추출)
  2. brightCanvas → Gaussian blur 수평 → blurCanvasH
  3. blurCanvasH → Gaussian blur 수직 → blurCanvasV
  4. sceneCanvas 원본 출력
  5. blurCanvasV additive blend 합성
```

**신규 파일:**
- `src/02_renderer/bloom.lua` — Canvas 4개 + 셰이더 2개 (threshold, 9-tap Gaussian)

**수정 파일:**
- `src/main.lua` — bloom require, init, draw 파이프라인 통합, F6 토글, resize 처리, debug watch

**파라미터:**
| 파라미터 | 값 | 설명 |
|----------|-----|------|
| threshold | 0.35 | 밝기 기준 (0-1) |
| intensity | 0.8 | glow 강도 |
| blurRadius | 3 | blur 커널 스케일 |

**키 바인딩:**
| 키 | 기능 |
|----|------|
| F6 | Bloom on/off 토글 (NEW) |
