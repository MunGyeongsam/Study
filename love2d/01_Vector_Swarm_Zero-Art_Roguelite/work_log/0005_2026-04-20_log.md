## 세션 6: Scene Stack 아키텍처 구현

### 구현 내용

#### sceneStack.lua — 스택 엔진 (01_core/)
- push/pop/replace/clear API
- update: top으로부터 아래로 transparent=true인 Scene까지 전파
- draw: top으로부터 drawBelow=true인 범위 수집 후 bottom-up 렌더
- input(keypressed/touch*): top Scene에만 전달

#### 6개 Scene 파일 (03_game/scenes/)
| Scene | drawBelow | transparent | 역할 |
|-------|-----------|-------------|-------|
| PlayScene | - | - | 전체 게임 루프 래핑 (ECS + 렌더 + 카메라 + BGM) |
| TitleScene | false | false | titleMenu 래핑, PLAY→replace(Play), UPGRADES→push(Upgrade) |
| PauseScene | true | false | pauseMenu 래핑, Continue→pop, Restart→restart, Menu→returnToTitle |
| LevelUpScene | true | false | levelUp 래핑, isActive() 해제 시 auto-pop |
| UpgradeScene | true | false | upgradeTree 래핑, isActive() 해제 시 auto-pop |
| GameOverScene | true | false | R→restart, U→push(Upgrade), ESC→returnToTitle |

#### main.lua 전면 리팩터
- 300줄+ 삭제: 모든 if-else 상태 분기 제거
- love.update(): `sceneStack:update(dt)` 한 줄
- love.draw(): `sceneStack:draw()` + 디버그 오버레이
- love.keypressed(): 디버그 키만 직접 처리, 나머지 sceneStack 위임
- love.touchpressed/moved/released: sceneStack 직접 위임
