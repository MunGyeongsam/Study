# 🎮 Vector Swarm — Play Guide

> 탄막 + 뱀서라이크 로그라이트. 적을 피하고, 처치하고, 레벨업하라!

---

## 게임 목표

- **생존**: 최대한 오래 살아남기 (Score = 생존 시간)
- **처치**: 플레이어 무기는 자동 조준 — 가장 가까운 적을 자동 발사
- **성장**: 적 처치 → XP 오브 수집 → 레벨업 → 3택 1 업그레이드 선택
- **회피**: 적 탄막을 피하면서 대쉬/포커스를 활용

---

## PC 조작

### 이동
| 키 | 동작 |
|----|------|
| `W` / `↑` | 위로 이동 |
| `S` / `↓` | 아래로 이동 |
| `A` / `←` | 왼쪽 이동 |
| `D` / `→` | 오른쪽 이동 |

> 대각선 이동 시 속도가 보정됩니다 (정규화)

### 액션
| 키 | 동작 | 설명 |
|----|------|------|
| `Shift` | **대쉬** | 이동 방향으로 순간이동 (2유닛), 무적 0.3초, 쿨타임 3초 |
| `Space` (홀드) | **포커스** | 시간 느려짐(40%), 판정 축소(40%), 이속 절반. 에너지 소모 |
| `R` | 재시작 | Game Over 상태에서만 |
| `ESC` | 종료 | |

### 레벨업 선택
| 키 | 동작 |
|----|------|
| `1` | 1번 업그레이드 선택 |
| `2` | 2번 업그레이드 선택 |
| `3` | 3번 업그레이드 선택 |

### 공격
- **자동 발사**: 범위(6유닛) 내 가장 가까운 적을 자동 조준·발사
- 조작 불필요 — 이동과 회피에 집중!

## 게임 메카닉

### 체력 (HP)
- 기본 HP: **5**
- 피격 시 **1.5초 무적** (깜빡임 이펙트)
- HP 0 → Game Over

### 대쉬
- 이동 방향으로 **2유닛** 순간이동
- 대쉬 중 **0.3초 무적**
- 쿨타임 **3초** (빨간 링 게이지로 표시)
- 입력 없이 대쉬 시 위쪽으로 이동

### 포커스 모드
- `Space` 홀드로 활성화
- 시간이 **40%** 속도로 느려짐 (슬로모션)
- 판정이 **60%** 축소 (탄막 회피 용이)
- 이동 속도 **50%** 감소
- 에너지 소모: **30/초** (최대 100)
- 에너지 회복: **15/초** (비활성 시)
- 보라색 아우라 + 축소 판정 원으로 시각 표시

### 무기 (자동)
- 초당 **4발** 발사
- 사거리 **6유닛** 내 가장 가까운 적 자동 조준
- 레벨업으로 탄수, 데미지, 발사속도 등 강화 가능

### XP & 레벨업
- 적 처치 시 **초록색 XP 오브** 드롭 (적 타입별 xpValue 상이)
- 플레이어 근처(**1.5유닛**)에서 자석처럼 끌려옴
- XP 충족 시 **레벨업** → 게임 일시정지 → **3개 옵션** 중 1개 선택

### 레벨업 옵션 (10종)

**무기 업그레이드:**
| 옵션 | 효과 |
|------|------|
| Fire Rate +20% | 공격 속도 증가 |
| Multi Shot +1 | 발사 탄수 +1 |
| Damage +1 | 탄당 데미지 +1 |
| Bullet Speed +25% | 탄속 증가 |
| Range +20% | 자동 조준 범위 확대 |

**패시브 스킬:**
| 옵션 | 효과 |
|------|------|
| Max HP +1 | 최대 체력 +1 (즉시 회복) |
| Move Speed +15% | 이동 속도 증가 |
| Magnet +30% | XP 오브 수집 범위 확대 |
| Dash CD -20% | 대쉬 쿨타임 감소 |
| Focus Energy +25% | 포커스 에너지 최대치 증가 |

### 적 타입
| 타입 | HP | XP | 색상 | 행동 | 탄막 |
|------|----|----|------|------|------|
