# 🎮 Vector Swarm — Play Guide

> 탄막 + 뱀서라이크 로그라이트. 적을 피하고, 처치하고, 레벨업하라!
> 마지막 갱신: 2026-04-24

---

## 게임 목표

- **생존**: 최대한 오래 살아남기 (Score = 생존 시간)
- **처치**: 플레이어 무기는 자동 조준 — 가장 가까운 적을 자동 발사
- **인런 성장**: 적 처치 → XP 오브 수집 → 레벨업 → 3택 1 업그레이드 선택
- **메타 성장**: 적/보스 처치 → Data Fragment 획득 → 영구 강화 트리 투자
- **회피**: 적 탄막을 피하면서 대쉬/포커스를 활용
- **해금**: 도전과제 달성 → 새 무기/캐릭터/패시브 해금

## 신 선택 (Deity Ritual)

게임 시작 시 **4개의 신** 중 1개를 선택한다. 선택한 신은 **패시브 스탯 보너스** + **확률 기반 시그니처 능력**을 런 전체에 걸쳐 부여한다.

| 신 | 곡선 | 패시브 | 시그니처 |
|----|------|--------|----------|
| **Rose** | Rose Curve | HP 회복 +15%, 이속 +5% | Graze 시 HP 1 회복 (100%) |
| **Cycloid** | Epicycloid (k=3) | 데미지 +10%, 발사속도 +5% | 20% 확률 크리티컬 (2배 데미지) |
| **Lemniscate** | Lemniscate (∞) | 쿨다운 -10%, XP +10% | 5% 확률 대시 쿨타임 즉시 초기화 |
| **Inferno** | Devil's Curve | 최대 HP +2, 무적시간 +0.2s | 10% 확률 범위 폭발 (주변 적 피해) |

### 선택 방법
- PC: `[↑/↓/←/→]` 또는 `[1/2/3/4]`로 선택 → `[Enter]` 확정
- 선택 화면에서 각 신의 곡선이 **스트로크 애니메이션**으로 그려진다

## 첫 플레이 튜토리얼

첫 번째 플레이(totalRuns == 0)에서만 자동으로 4단계 인게임 힌트가 표시된다.

| 단계 | 메시지 | 트리거 | 완료 조건 |
|------|--------|--------|-----------|
| 1 | **MOVE** — WASD to move | 게임 시작 0.5초 후 | 이동 입력 감지 |
| 2 | **DASH** — SHIFT to dash | 1단계 완료 1초 후 | 대쉬 입력 감지 |
| 3 | **FOCUS** — SPACE to focus | 2단계 완료 1초 후 | 포커스 입력 감지 |
| 4 | **AUTO** — Auto-attack nearby | 3단계 완료 1초 후 | 3초 자동 대기 |

- 각 힌트 표시 시 **슬로모션** 적용 (timeScale 0.15) — 탄막 속에서도 안전하게 읽을 수 있음
- **글리치 텍스트** 이펙트 — Zero-Art 스타일로 ASCII 깨짐 + 네온 발광
- 4단계 모두 완료 시 `tutorialDone` 세이브 → 이후 플레이에서는 표시 안 됨

## 게임 메카닉

### 체력 (HP)
- 기본 HP: **5**
- 피격 시 **1.5초 무적** (깜빡임 이펙트)
- HP 0 → Game Over

### 대쉬
- 이동 방향으로 **2유닛** 순간이동
- 대쉬 중 **0.3초 무적**
- 쿨타임 **3초** (빨간 링 게이지로 표시)
- 입력 없이 대쉬 시 위쪽으로 이동

### 포커스 모드
- `Space` 홀드로 활성화
- 시간이 **40%** 속도로 느려짐 (슬로모션)
- 판정이 **60%** 축소 (탄막 회피 용이)
- 이동 속도 **50%** 감소
- 에너지 소모: **30/초** (최대 100)
- 에너지 회복: **15/초** (비활성 시)
- 보라색 아우라 + 축소 판정 원으로 시각 표시

### 무기 (자동)
- 초당 **4발** 발사
- 사거리 **6유닛** 내 가장 가까운 적 자동 조준
- 레벨업으로 탄수, 데미지, 발사속도 등 강화 가능

### XP & 레벨업
- 적 처치 시 **초록색 XP 오브** 드롭 (적 타입별 xpValue 상이)
- 플레이어 근처(**1.5유닛**)에서 자석처럼 끌려옴
- XP 충족 시 **레벨업** → 게임 일시정지 → **3개 옵션** 중 1개 선택
- XP 스케일링: 후반 스테이지일수록 XP 드롭량 증가 (스테이지당 +15%)

### 레벨업 옵션 (10종)

**무기 업그레이드:**
| 옵션 | 효과 |
|------|------|
| Fire Rate +20% | 공격 속도 증가 |
| Multi Shot +1 | 발사 탄수 +1 |
| Damage +1 | 탄당 데미지 +1 |
| Bullet Speed +25% | 탄속 증가 |
| Range +20% | 자동 조준 범위 확대 |

**패시브 스킬:**
| 옵션 | 효과 |
|------|------|
| Max HP +1 | 최대 체력 +1 (즉시 회복) |
| Move Speed +15% | 이동 속도 증가 |
| Magnet +30% | XP 오브 수집 범위 확대 |
| Dash CD -20% | 대쉬 쿨타임 감소 |
| Focus Energy +25% | 포커스 에너지 최대치 증가 |

> **감쇠 스택**: 같은 업그레이드를 반복 선택하면 효과가 줄어든다 (매회 70%). 예: Fire Rate 1회 +20% → 2회 +14% → 3회 +10%
> Multi Shot 과 Max HP는 고정값(+1)이라 감쇠 없음.

### 적 타입
| 타입 | HP | XP | 색상 | 행동 | 탄막 |
|------|----|----|------|------|------|
| **Basic** | 3 | 2 | 🔴 빨강 | 직선 하강 | 원형 6발 |
| **Spiral** | 5 | 5 | 🟣 보라 | 원형 궤도 | 나선형 4발 |
| **Aimed** | 2 | 3 | 🟡 노랑 | 플레이어 추적 | 조준 3발 |
| **Wave** | 4 | 3 | 🟢 초록 | 비스듬 하강 | 파동형 5발 |

### 스테이지 진행
- 웨이브의 적을 전멸하면 **스테이지 클리어**
- 스테이지가 올라갈수록 적 수, HP, 속도, 탄속 증가
- **Stage 3, 6, 9, 12, 15**마다 보스 등장
- Stage 16+ 매 3스테이지마다 보스 반복 (무한 스테이지)

### 보스
| 보스 | 스테이지 | HP | 페이즈 | 특징 |
|------|:---:|:---:|:---:|------|
| **NULL** | 3 | 50 | 2 | 기본 탄막 (튜토리얼) |
| **STACK** | 6 | 100 | 3 | circle + aimed 조합 |
| **HEAP** | 9 | 160 | 3 | 텔레포트 + 점멸 경고 |
| **RECURSION** | 12 | 200 | 3 | 공전 이동 + 미니언 소환 |
| **OVERFLOW** | 15 | 300 | 4 | 전 패턴 순환 + 페이즈별 색상/이동 전환 |

- 보스 처치 시: 탄막 전소거 + XP 폭발 + HP/대쉬/포커스 회복
- 페이즈 전환: HP가 일정 비율 이하로 떨어지면 탄막 패턴 변경 + 강화

## 메타 성장 시스템

### Data Fragment (영구 재화)
- 적/보스 처치 시 **Data Fragment** 획득
- 게임오버 시 누적 Fragment가 세이브에 확정
- 세계관: 데이터스피어의 파편 — 적이 해체되며 남기는 정보 조각

### 영구 강화 트리 (Upgrade Tree)

타이틀 메뉴 또는 게임오버 화면에서 `UPGRADES`로 접근.
Fragment를 소비하여 영구적인 능력치 강화.

| 카테고리 | 강화 항목 | 최대 | 단계당 효과 |
|---------|----------|:----:|-----------|
| **Attack** | Core Damage | 3 | +1 기본 데미지 |
| | Overclock (Fire Rate) | 3 | +15% 발사속도 |
| | Long Reach (Range) | 3 | +20% 자동조준 범위 |
| **Defense** | Reinforced Core (Max HP) | 3 | +1 최대 HP |
| | Phase Shield (iFrame) | 3 | +0.3초 무적시간 |
| | Quick Phase (Dash CD) | 3 | -15% 대시 쿨타임 |
| **Utility** | Data Magnet | 3 | +30% 수집범위 |
| | Turbo (Move Speed) | 3 | +10% 이동속도 |
| | Data Miner (Fragment) | 3 | +5% Fragment 드롭률 |

> 영구 강화는 기본 스탯에 합산 → 인런 레벨업은 그 위에 추가로 쌓임
> 상세 수치와 밸런싱 근거는 `21_PROGRESSION_DESIGN.md` 참조

### 도전과제 & 해금

`ACHIEVEMENTS` 메뉴에서 진행 상황 확인 가능.

| 조건 | 해금 |
|------|------|
| Stage 3 최초 클리어 | 무기: Spread Shot (부채꼴 발사) |
| Stage 6 최초 클리어 | 무기: Piercing (관통탄) |
| 보스 5종 전부 처치 | 캐릭터: "Debugger" (작은 히트박스, 낮은 HP) |
| 누적 1000 처치 | 캐릭터: "Compiler" (높은 HP, 느린 이동) |
| 누적 Fragment 500 | 패시브: Start Boost (시작 시 랜덤 업그레이드 1개) |

- 도전과제 달성 시 게임오버 화면에 **NEW!** 알림 표시
- 진행도 바로 진행률 확인 가능
